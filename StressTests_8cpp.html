Detailed Description

Concurrency and Stability tests for the ThreadSafeOrderBook.

This file performs "Hammer Tests" to verify the SpinLock implementation. It spawns multiple threads that aggressively hammer the engine with orders.

  • Pass Condition: The program does not Segfault or throw exceptions.
  • Fail Condition: Memory corruption (Segfault) due to race conditions.

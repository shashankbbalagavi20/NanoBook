Detailed Description

NanoBook CLI - Live Trading Dashboard.

This is the visual entry point for the project. It runs a continuous simulation of market data and renders a "Bloomberg Terminal" style TUI (Text User Interface).

  • NOTE: This is for demonstration purposes. For actual performance measurements, see 'src/benchmark.cpp'.

void clearScreen ( )

Clears the console screen using ANSI escape codes.

  • \033[2J : Clears the entire screen.
  • \033[1;1H : Moves the cursor to the top-left corner (Row 1, Col 1). This creates a "static" dashboard effect instead of a scrolling log.
